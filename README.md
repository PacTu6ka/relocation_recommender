# Relocation Recommender

ML-система для персонализированных рекомендаций по переезду на основе предпочтений пользователя.

## Описание

Проект анализирует данные о городах и регионах (стоимость жизни, климат, рынок труда, инфраструктура) и подбирает наиболее подходящие локации для переезда с учётом индивидуальных приоритетов пользователя.

## Архитектура

- **Кластеризация** — HDBSCAN + UMAP для группировки похожих локаций
- **Рекомендации** — гибридный подход: коллаборативная фильтрация + RL (Stable Baselines3 + Gymnasium)
- **Объяснимость** — SHAP-значения для интерпретации рекомендаций
- **Интерфейс** — интерактивный дашборд на Streamlit с визуализацией на Plotly

## Структура проекта

```
relocation_recommender/
├── data/
│   ├── raw/            # исходные данные (парсинг, API)
│   └── processed/      # очищенные и трансформированные данные
├── notebooks/          # EDA и эксперименты
├── src/
│   ├── preprocessor.py # очистка и feature engineering
│   ├── clusterer.py    # кластеризация локаций
│   ├── recommender.py  # модель рекомендаций
│   └── explainer.py    # объяснимость предсказаний
├── app/
│   └── streamlit_app.py  # веб-интерфейс
└── requirements.txt
```

## Установка
